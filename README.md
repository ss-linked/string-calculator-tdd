# String calculator TDD kata

Hello and welcome to my solution for the string calculator kata!

## Stack

- **Language**: Javascript
- **Build tool**: Vite
- **Tests**: Vitest
- **Styling**: PicoCSS

## Notes

- **Node version**: v22.8.0

## Instructions

- Clone the repo
- Run `npm install`
- To run tests, run `npm test`
- To run a local server, run `npm start`
